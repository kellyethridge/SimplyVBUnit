** SimplyVBUnit 4.0 **

-- This version breaks backward compatibility when implementing ITestCaseData. The new interface is ITestCaseSource.
-- Starting with version 4, SimplyVBUnit supports categorizing unit tests. This can be achieved by implementing ICategorizable.

* Before running any unit tests, be sure to set the VB6 IDE Error Trapping to 'Break on Unhandled Errors'. This setting can be found in Tools->Options->General Tab. *
